# brython-document

brython中文文档

[Brython](http://www.brython.info)的设计目的是取代`javascript`作为Web的脚本语言。
因此，它是一个`python3` 实现（您可以通过一个Web控制台将它作为一个测试驱动），适合于HTML5环境，也就是说，它实现了一个DOM对象和事件的接口。
